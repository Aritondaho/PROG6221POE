# BloomGuard Bot

BloomGuard Bot is a cybersecurity awareness chatbot created using C# Windows Forms.

## Features

- Cybersecurity awareness chatbot
- Interactive chatbot responses
- Audio greeting feature
- ASCII art welcome screen
- Friendly user interface
- GitHub version control with commits

## Topics Covered

The chatbot can help users learn about:

- Phishing
- Password safety
- Online scams
- Privacy protection
- Public Wi-Fi safety
- Two-factor authentication (2FA)

## Technologies Used

- C#
- Windows Forms
- .NET
- Visual Studio
- GitHub

## How to Run

1. Open the project in Visual Studio
2. Build the solution
3. Run the application
4. Start chatting with BloomGuard Bot
